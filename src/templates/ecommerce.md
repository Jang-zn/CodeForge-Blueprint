## 예시

- 이런 서비스예요: 동네 수제 먹거리 판매자(빵집, 반찬가게, 수제청 등)와 인근 구매자를 연결하는 지역 기반 마켓 플랫폼입니다. 반경 5km 이내 판매자를 우선 노출하고, 당일 픽업 또는 동네 배달을 지원합니다.
- 주요 화면: 홈(내 위치 기반 판매자 지도 + 리스트), 판매자 상점(상품 목록 + 픽업 시간), 장바구니·주문, 주문 현황(실시간 준비 상태), 판매자용 주문 관리 화면
- 사용 흐름: 구매자가 위치 허용 후 주변 판매자 탐색 → 상품 담기 → 픽업 시간 선택 후 결제 → 판매자가 앱에서 주문 확인 후 준비 완료 알림 → 구매자가 픽업 → 후기 작성
- 경쟁 서비스와 다른 점: 쿠팡·배민은 대형 브랜드 중심이라 소규모 수제 판매자가 입점하기 어렵습니다. 우리 서비스는 개인 판매자가 5분 안에 상점을 개설할 수 있고, 지역 커뮤니티 성격을 강조합니다.
- 처음에 꼭 필요한 기능: 판매자 상점 개설(사진·상품 등록), 위치 기반 탐색, 픽업 시간 선택, 토스페이먼츠 결제, 구매자·판매자 간 알림, 후기 작성
