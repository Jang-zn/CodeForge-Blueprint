## 예시

- 이런 서비스예요: 개발자가 공부 기록을 마크다운으로 작성하고 구독자와 공유하는 개발자 전용 블로그·뉴스레터 플랫폼입니다. 글쓴이는 코드 블록·다이어그램을 쉽게 삽입하고, 독자는 시리즈 단위로 구독할 수 있습니다.
- 주요 화면: 피드(팔로우한 작가 최신 글), 에디터(마크다운 + 코드 하이라이트 + Mermaid 다이어그램), 시리즈 관리, 구독자 통계(조회수·구독자 수·이탈률), 뉴스레터 발송 설정
- 사용 흐름: 작가가 마크다운 에디터에서 글 작성 → 시리즈에 포함 또는 단독 발행 → 구독자에게 이메일 발송 → 독자가 피드 또는 이메일 링크로 읽기 → 댓글·좋아요 → 작가가 통계 확인
- 경쟁 서비스와 다른 점: 티스토리·velog는 SEO 위주이고 뉴스레터 기능이 없습니다. Substack은 코드 블록 지원이 빈약합니다. 우리 서비스는 개발자 글쓰기 경험(코드·다이어그램)과 뉴스레터 구독을 동시에 지원합니다.
- 처음에 꼭 필요한 기능: 마크다운 에디터(코드 블록·Mermaid), 발행·비공개 전환, 시리즈 묶기, 이메일 구독·발송, 조회수·구독자 통계
